# 🍔 Food Delivery Web App

A modern **Food Delivery Web Application** built using **React.js and Vite** that allows users to browse food items, add them to a cart, and place orders through a simple and responsive interface.

This project focuses on building a **clean UI, reusable components, and dynamic cart functionality**, similar to popular food delivery platforms.

---

## 🚀 Live Demo

🔗 https://mahesh09kumar.github.io/food_delivery_web_app/

---

## 📌 Features

- 🍕 Browse food items with images and prices
- 🛒 Add or remove items from cart
- 🔢 Increase or decrease item quantity
- 💰 Dynamic cart total calculation
- 📱 Fully responsive design
- 🔐 Login popup interface
- 🧭 Navigation using React Router

## ⚙️ Installation & Setup

Clone the repository

```
git clone https://github.com/Mahesh09kumar/food_delivery_web_app.git
```

Navigate to project folder

```
cd food_delivery_web_app
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
